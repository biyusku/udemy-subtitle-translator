## ✨ विशेषताएं

- 🌍 **बहुभाषी समर्थन** — प्रीसेट भाषाओं में से चुनें या कोई भी ISO/BCP47 कोड दर्ज करें
- 🧠 **स्मार्ट स्रोत पहचान** — `video.textTracks`, कैप्शन DOM या ट्रांसक्रिप्ट पैनल से पढ़ता है
- 🖥️ **फुलस्क्रीन तैयार** — ट्रांसक्रिप्ट कैश पैनल बंद होने के बाद भी अनुवाद जारी रखता है
- 👁️ **मूल सब्टाइटल छुपाएं** — केवल अनुवादित परत दिखाएं
- ⚡ **अनुवाद कैश** — दोहराई गई पंक्तियां बिना नई रिक्वेस्ट के तुरंत मिलती हैं
- 🔌 **दो प्रदाता** — Google GTX (बिना सेटअप) या अपना LibreTranslate सर्वर
- 🎯 **पहली बार सेटअप विज़ार्ड** — एक सवाल में डिफ़ॉल्ट भाषा तय करें
- 🛠️ **कोई बिल्ड स्टेप नहीं** — शुद्ध JS, सीधे लोड होता है

---

## 🚀 इंस्टॉलेशन

### डेवलपर मोड (मैन्युअल)

1. इस रिपो को क्लोन करें या ZIP के रूप में डाउनलोड करें
2. Chrome में **`chrome://extensions`** खोलें
3. ऊपर दाईं ओर **Developer mode** चालू करें
4. **Load unpacked** पर क्लिक करें
5. **`extension/`** फ़ोल्डर चुनें

> Chrome Web Store पर जल्द आ रहा है।

## 🔧 यह कैसे काम करता है

```
Udemy लेक्चर पेज
       │
       ▼
 कंटेंट स्क्रिप्ट  (content.js)
   सक्रिय सब्टाइटल टेक्स्ट पहचानता है
       │
       ▼
 बैकग्राउंड वर्कर  (background.js)
   चुने हुए प्रदाता से अनुवाद करता है
   दोहराई गई पंक्तियों को कैश करता है
       │
       ▼
 वीडियो के ऊपर परत के रूप में दिखाता है
```

## 🌐 अनुवाद प्रदाता

| प्रदाता | सेटअप | नोट्स |
|---|---|---|
| **Google GTX** | कोई नहीं | डिफ़ॉल्ट। कोई API key नहीं चाहिए। |
| **LibreTranslate** | Endpoint URL | अपना या सार्वजनिक सर्वर। पूरी गोपनीयता। |

## 🎛️ उपयोग

1. किसी भी Udemy लेक्चर पेज पर जाएं
2. एक्सटेंशन आइकन पर क्लिक करें
3. पहली बार — अपनी भाषा चुनें
4. वीडियो पर कैप्शन चालू करें या ट्रांसक्रिप्ट पैनल खोलें
5. **Extension enabled** चालू रखें
6. अनुवादित सब्टाइटल वीडियो के ऊपर दिखेंगे
