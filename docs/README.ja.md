## ✨ 機能

- 🌍 **多言語対応** — プリセット言語から選択するか、ISO/BCP47コードを入力
- 🧠 **スマートなソース検出** — `video.textTracks`、字幕DOM、またはトランスクリプトパネルから読み取り
- 🖥️ **フルスクリーン対応** — トランスクリプトキャッシュにより、パネルが閉じた後も翻訳を継続
- 👁️ **元の字幕を非表示** — 翻訳オーバーレイのみ表示
- ⚡ **翻訳キャッシュ** — 繰り返し出現する行を即座に提供
- 🔌 **2つのプロバイダー** — Google GTX（設定不要）または独自LibreTranslateサーバー
- 🎯 **初回セットアップウィザード** — 一問一答でデフォルト言語を設定
- 🛠️ **ビルド不要** — 純粋なJS、`Load unpacked`で直接読み込み可能

## 🚀 インストール

### デベロッパーモード（手動）

1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで **`chrome://extensions`** を開く
3. 右上の **デベロッパーモード** を有効化
4. **パッケージ化されていない拡張機能を読み込む** をクリック
5. **`extension/`** フォルダを選択

> Chrome ウェブストア版は近日公開予定。

## 🔧 仕組み

```
Udemyの講義ページ
       │
       ▼
 コンテンツスクリプト  (content.js)
   アクティブな字幕テキストを検出
       │
       ▼
 バックグラウンドワーカー  (background.js)
   選択したプロバイダーで翻訳
   繰り返し行をキャッシュ
       │
       ▼
 動画の上にオーバーレイとして表示
```

1. コンテンツスクリプトがページのアクティブな字幕テキストを監視。
2. 新しい行ごとにバックグラウンドサービスワーカーに送信。
3. ワーカーがテキストを翻訳し結果をキャッシュ。
4. 翻訳字幕が動画の上に直接表示される。

## 🌐 翻訳プロバイダー

| プロバイダー | セットアップ | 備考 |
|---|---|---|
| **Google GTX** | 不要 | デフォルト。APIキー不要。 |
| **LibreTranslate** | エンドポイントURL | 自前またはパブリックサーバー。プライバシー完全制御。 |

---

## 🎛️ 使い方

1. Udemyの講義ページを開く
2. ツールバーの拡張機能アイコンをクリック
3. 初回起動時 — 言語を選択
4. 動画の字幕をオンにするか、トランスクリプトパネルを開く
5. **拡張機能を有効** をオンのままにする
6. 翻訳字幕が動画の上に表示される

---

## ⚠️ 制限事項

- 字幕またはトランスクリプトがある講義のみ対応
- ライブ音声認識は非対応
- 翻訳品質はプロバイダーと言語ペアによって異なる

---

## 🔒 プライバシー

この拡張機能は字幕/トランスクリプトのテキストを選択したプロバイダーに送信する場合があります。閲覧履歴、個人データ、Udemyの認証情報は一切収集しません。

詳細は [PRIVACY.md](../PRIVACY.md) をご覧ください。
