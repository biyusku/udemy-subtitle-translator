## ✨ คุณสมบัติ

- 🌍 **รองรับหลายภาษา** — เลือกจากภาษาที่ตั้งไว้ล่วงหน้า หรือกรอกรหัส ISO/BCP47
- 🧠 **การตรวจจับแหล่งที่มาอัจฉริยะ** — อ่านจาก `video.textTracks`, DOM คำบรรยาย หรือแผงสคริปต์
- 🖥️ **พร้อมสำหรับเต็มหน้าจอ** — แคชสคริปต์รักษาการแปลให้ทำงานต่อ
- 👁️ **ซ่อนคำบรรยายต้นฉบับ** — แสดงเฉพาะชั้นที่แปลแล้ว
- ⚡ **แคชการแปล** — ประโยคซ้ำจะถูกส่งทันทีโดยไม่ต้องร้องขอใหม่
- 🔌 **สองผู้ให้บริการ** — Google GTX (ไม่ต้องตั้งค่า) หรือเซิร์ฟเวอร์ LibreTranslate ของคุณเอง
- 🎯 **ตัวช่วยการตั้งค่าครั้งแรก** — คำถามเดียวเพื่อกำหนดภาษาเริ่มต้น
- 🛠️ **ไม่มีขั้นตอนการ build** — JS ล้วนๆ โหลดได้โดยตรง

## 🚀 การติดตั้ง

### โหมดนักพัฒนา (แบบแมนวล)

1. โคลนหรือดาวน์โหลดที่เก็บนี้เป็น ZIP
2. เปิด **`chrome://extensions`** ใน Chrome
3. เปิดใช้ **โหมดนักพัฒนา** (มุมขวาบน)
4. คลิก **โหลดส่วนขยายที่แตกไฟล์แล้ว**
5. เลือกโฟลเดอร์ **`extension/`**

> เวอร์ชัน Chrome Web Store เร็วๆ นี้

## 🌐 ผู้ให้บริการการแปล

| ผู้ให้บริการ | การตั้งค่า | หมายเหตุ |
|---|---|---|
| **Google GTX** | ไม่มี | ค่าเริ่มต้น ไม่ต้องใช้ API key |
| **LibreTranslate** | URL ของ Endpoint | เซิร์ฟเวอร์ส่วนตัวหรือสาธารณะ ควบคุมความเป็นส่วนตัวได้เต็มที่ |

## 🎛️ การใช้งาน

1. ไปที่หน้าบทเรียน Udemy ใดก็ได้
2. คลิกไอคอนส่วนขยายในแถบเครื่องมือ
3. เมื่อเปิดครั้งแรก — เลือกภาษาของคุณ
4. เปิดคำบรรยายในวิดีโอหรือเปิดแผงสคริปต์
5. เปิด **ส่วนขยายที่เปิดใช้งาน** ไว้
6. คำบรรยายที่แปลแล้วจะปรากฏบนวิดีโอ

## ⚠️ ข้อจำกัด

- ทำงานได้เฉพาะกับคอร์สที่มีคำบรรยายหรือสคริปต์อยู่แล้ว
