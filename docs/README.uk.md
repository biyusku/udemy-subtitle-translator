## ✨ Можливості

- 🌍 **Підтримка багатьох мов** — вибирайте з готового списку або вводьте код ISO/BCP47
- 🧠 **Розумне визначення джерела** — читає з `video.textTracks`, DOM субтитрів або панелі транскрипту
- 🖥️ **Підтримка повного екрана** — кеш транскрипту зберігає переклади активними
- 👁️ **Приховати оригінальні субтитри** — показувати лише перекладений оверлей
- ⚡ **Кеш перекладів** — рядки, що повторюються, видаються миттєво
- 🔌 **Два провайдери** — Google GTX (без налаштування) або власний сервер LibreTranslate
- 🎯 **Майстер першого запуску** — одне запитання для вибору мови за замовчуванням
- 🛠️ **Без кроку збірки** — чистий JS, завантажується напряму

## 🚀 Встановлення

### Режим розробника (вручну)

1. Клонуйте або завантажте цей репозиторій як ZIP
2. Відкрийте **`chrome://extensions`** у Chrome
3. Увімкніть **Режим розробника** (правий верхній кут)
4. Натисніть **Завантажити розпаковане розширення**
5. Виберіть папку **`extension/`**

> Версія для Chrome Web Store незабаром.

## 🔧 Як це працює

```
Сторінка лекції Udemy
       │
       ▼
 Контентний скрипт  (content.js)
   Виявляє активний текст субтитрів
       │
       ▼
 Фоновий воркер  (background.js)
   Перекладає через обраного провайдера
   Кешує рядки, що повторюються
       │
       ▼
 Оверлей додається поверх відео
```

## 🌐 Провайдери перекладу

| Провайдер | Налаштування | Примітки |
|---|---|---|
| **Google GTX** | Немає | За замовчуванням. Ключ API не потрібен. |
| **LibreTranslate** | URL ендпоінту | Власний або публічний сервер. Повний контроль конфіденційності. |

---

## 🎛️ Використання

1. Перейдіть на будь-яку сторінку лекції Udemy
2. Натисніть на значок розширення
3. При першому запуску — виберіть мову
4. Увімкніть субтитри у відео або відкрийте панель транскрипту
5. Тримайте **Розширення увімкнено**
6. Перекладені субтитри з'являться поверх відео

---

## ⚠️ Обмеження

- Працює лише з курсами, які вже мають субтитри або транскрипт
- Не виконує живе перетворення мовлення в текст
- Якість перекладу залежить від провайдера та мовної пари

---

## 🔒 Конфіденційність

Розширення може надсилати текст субтитрів обраному провайдеру. Історія перегляду, особисті дані та облікові дані Udemy не збираються.
