## ✨ Возможности

- 🌍 **Поддержка множества языков** — выбирайте из готового списка или вводите код ISO/BCP47
- 🧠 **Умное определение источника** — читает из `video.textTracks`, DOM субтитров или панели транскрипта
- 🖥️ **Поддержка полного экрана** — кэш транскрипта сохраняет переводы активными после закрытия панели
- 👁️ **Скрыть оригинальные субтитры** — показывать только переведённый оверлей
- ⚡ **Кэш переводов** — повторяющиеся строки отдаются мгновенно
- 🔌 **Два провайдера** — Google GTX (без настройки) или собственный сервер LibreTranslate
- 🎯 **Мастер первого запуска** — один вопрос для выбора языка по умолчанию
- 🛠️ **Без шага сборки** — чистый JS, загружается напрямую

## 🚀 Установка

### Режим разработчика (вручную)

1. Клонируйте или скачайте этот репозиторий как ZIP
2. Откройте **`chrome://extensions`** в Chrome
3. Включите **Режим разработчика** (правый верхний угол)
4. Нажмите **Загрузить распакованное расширение**
5. Выберите папку **`extension/`**

> Версия для Chrome Web Store скоро будет доступна.

## 🔧 Как это работает

```
Страница лекции Udemy
       │
       ▼
 Контентный скрипт  (content.js)
   Обнаруживает активный текст субтитров
       │
       ▼
 Фоновый воркер  (background.js)
   Переводит через выбранного провайдера
   Кэширует повторяющиеся строки
       │
       ▼
 Оверлей добавляется поверх видео
```

1. Контентный скрипт отслеживает активные субтитры на странице.
2. Каждая новая строка отправляется в фоновый сервис-воркер.
3. Воркер переводит текст и кэширует результат.
4. Переведённый субтитр отображается прямо поверх видео.

## 🌐 Провайдеры перевода

| Провайдер | Настройка | Заметки |
|---|---|---|
| **Google GTX** | Нет | По умолчанию. Ключ API не требуется. |
| **LibreTranslate** | URL эндпоинта | Собственный или публичный сервер. Полный контроль конфиденциальности. |

## 🎛️ Использование

1. Перейдите на страницу любой лекции Udemy
2. Нажмите на значок расширения
3. При первом запуске — выберите язык
4. Включите субтитры на видео или откройте панель транскрипта
5. Держите **Расширение включено**
6. Переведённые субтитры появятся поверх видео

---

## ⚠️ Ограничения

- Работает только с курсами, у которых есть субтитры или транскрипт
- Не выполняет живое преобразование речи в текст
- Качество перевода зависит от провайдера и языковой пары

---

## 🔒 Конфиденциальность

Расширение может отправлять текст субтитров выбранному провайдеру. История просмотров, личные данные и учётные данные Udemy не собираются.

Читайте [PRIVACY.md](../PRIVACY.md) для подробностей.
