# Security Policy

## Reporting a vulnerability

If you discover a security issue in this project, do not post sensitive details in a public issue.

Instead, share:

- a concise description of the problem
- the impact
- reproduction steps
- any logs or screenshots that help confirm it

Until a dedicated security contact is added, keep reports private in the channel where this project is maintained.

## Scope notes

Particular care should be taken around:

- script behavior on Udemy lecture pages
- fullscreen handling
- cross-origin translation requests
- privacy of caption and transcript text
